## Run Locally

Clone the project

```bash
  git clone https://github.com/Majdi-Akrmi/My_Portfolio.git
```

Go to the project directory

```bash
  cd My_Portfolio
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
